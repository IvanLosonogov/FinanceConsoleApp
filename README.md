# FinanceConsoleApp
Консольное приложение для управления личными финансами с системой авторизации пользователей, учетом доходов/расходов, бюджетированием и экспортом отчетов.
## Требования
- Java 11 или выше
- Maven 3.6+
## Сборка и запуск
mvn clean compile

mvn exec:java -Dexec.mainClass="com.myfinance.Main"

## Первый запуск
При первом запуске будет создана папка data/ для хранения информации
Будет создан тестовый пользователь: логин: test, пароль: test123
Можно использовать тестового пользователя или зарегистрировать нового

## Пример сеанса работы
```
============================================================
          ПРИЛОЖЕНИЕ ДЛЯ УПРАВЛЕНИЯ ФИНАНСАМИ
============================================================
Введите '10' в главном меню для получения справки
============================================================

========================================
АВТОРИЗАЦИЯ
========================================
1. Войти
2. Зарегистрироваться
3. Выход из приложения
========================================
Выберите действие: 1
Введите логин: test
Введите пароль: test123

✅ УСПЕХ: Добро пожаловать, test!

========================================
ГЛАВНОЕ МЕНЮ
========================================
 1. Добавить доход
 2. Добавить расход
 3. Установить бюджет
 4. Показать статистику
 5. Показать транзакции
 6. Показать бюджеты
 7. Создать категорию
 8. Экспорт отчета (CSV)
 9. Редактировать бюджет
10. Справка
 0. Выход
========================================
Введите номер команды: 1
Введите сумму дохода: 50000
Введите категорию дохода: Зарплата
Введите описание (необязательно): Зарплата за декабрь
УСПЕХ: Доход добавлен
```

## Структура проекта
```
src/main/java/com/myfinance/
├── model/                          # Доменные сущности
│   ├── User.java                   # Пользователь
│   ├── Wallet.java                 # Кошелек пользователя
│   ├── Transaction.java            # Абстрактная транзакция
│   ├── TransactionType.java        # Тип операции (enum)
│   ├── Income.java                 # Доход
│   ├── Expense.java                # Расход
│   ├── Category.java               # Категория операции
│   └── Budget.java                 # Бюджет на категорию
│
├── repository/                     # Уровень доступа к данным
│   ├── UserRepository.java         # Хранилище пользователей
│   └── WalletRepository.java       # Хранилище кошельков
│
├── service/                        # Бизнес-логика
│   ├── AuthService.java            # Авторизация и регистрация
│   ├── WalletService.java          # Управление кошельком
│   └── NotificationService.java    # Оповещения пользователя
│
├── cli/                           # Пользовательский интерфейс
│   ├── CommandExecutor.java        # Обработчик команд
│   └── ConsolePrinter.java         # Вывод информации в консоль
│
├── util/                          # Вспомогательные классы
│   ├── FileStorage.java            # Работа с файлами
│   └── Validator.java              # Валидация данных
│
└── Main.java                      # Точка входа в приложение
```
## Список команд

| №  | Команда               | Описание                                   | Пример                 |
|----|-----------------------|---------------------------------------------|------------------------|
| 1  | Добавить доход        | Зафиксировать поступление денег              | 50000 Зарплата "Аванс" |
| 2  | Добавить расход       | Зафиксировать трату                          | 1500 Еда "Продукты"    |
| 3  | Установить бюджет     | Установить лимит на категорию                | Еда 10000              |
| 4  | Показать статистику   | Обзор финансового состояния                  | —                      |
| 5  | Показать транзакции   | История всех операций                        | —                      |
| 6  | Показать бюджеты      | Текущие бюджеты и их использование           | —                      |
| 7  | Создать категорию     | Добавить новую категорию                     | Спорт расход           |
| 8  | Экспорт отчета        | Экспорт данных в CSV файл                    | report_december        |
| 9  | Редактировать бюджет  | Изменить существующий бюджет                 | Еда 12000              |
| 10 | Справка               | Показать список команд                       | —                      |
| 0  | Выход                 | Выйти из системы (данные сохраняются)        | —                      |
